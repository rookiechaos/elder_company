# Presentation Guide

## File Structure

1. **PRESENTATION_JP.md** — Full version (detailed content, design suggestions)
2. **PRESENTATION_JP_SIMPLE.md** — Simple version (key points per slide)

## Usage

### Method 1: Copy to PowerPoint/Google Slides
1. Open `PRESENTATION_JP_SIMPLE.md`
2. Copy each slide's content
3. Paste into PowerPoint or Google Slides
4. Apply design template

### Method 2: Customize from Full Version
1. Open `PRESENTATION_JP.md`
2. Review detailed slide content
3. Customize as needed
4. Transfer to PowerPoint

## Design Template

### Recommended Color Palette
```
Main:    #F5E6D3 (beige)
Accent:  #4A90A4 (light blue)
Emphasis:#E8A87C (orange)
Background: #FFFFFF or #F9F9F9
Text:    #333333
```

### Recommended Fonts
- **Japanese**: Yu Gothic Medium, Meiryo
- **English**: Arial Bold, Helvetica
- **Sizes**: Title 32–36pt | Subtitle 24–28pt | Body 20–24pt | Min 18pt

### Layout Principles
1. One message per slide
2. Bullet lists: 3–5 items max
3. Generous whitespace
4. High-quality images

## Recommended Visuals

### Screenshots
- Dashboard (stats, activity library, user management)
- Translation (real-time, terminology library, voice I/O)
- Activity library (categories, recommendations, customization)
- Mobile app (phone and tablet views)

### Icons
Collaboration, communication, activities, cooperation, statistics, security, performance

## Presentation Tips

### Storytelling
- Problem → solution flow
- Concrete usage scenarios
- Emotional value alongside data

### Data & Evidence
- Highlight metrics (30–50% improvement)
- Before/after comparisons
- Case studies when available

### Interaction
- Prompt questions, live demo if possible, read audience feedback

### Time Management
- 15 slides ≈ 20–30 minutes
- Q&A: 10–15 minutes
- Demo: 5–10 minutes (optional)

## FAQ Preparation

**Q: Is it secure?**
- Encrypted in transit and at rest, Japan APPI compliant, regular audits, access control and audit logs

**Q: What does it cost?**
- Basic ¥5,000/mo (10 users, 1,000 translations) | Professional ¥20,000/mo (50 users, 10,000) | Enterprise custom | Early adopter: 50% off first 10 facilities

**Q: Is implementation difficult?**
- Guided setup, comprehensive training, 1–3 month free trial, ongoing support

**Q: Can it integrate with existing systems?**
- API available, custom integration options, data export/import

**Q: What support is available?**
- Setup assistance, staff training, email/phone/online support, regular updates

## Customization Tips

- Adjust emphasis by facility size and type (nursing home, day service, etc.)
- Prepare live demo with relatable sample data; keep screenshots as backup
- Add case studies, technical appendix, and FAQ sheet as needed

## Follow-Up After Presentation

1. Email follow-up within 24 hours
2. Schedule demos for interested parties
3. Share presentation materials
4. Answer all questions

# プレゼンテーション資料の使い方

1. **PRESENTATION_JP.md** — 完全版（詳細な内容、デザイン提案）

## 使い方

### 方法1: PowerPoint/Google Slides に直接コピー

1. `PRESENTATION_JP_SIMPLE.md` を開く
2. 各スライドの内容をコピー

4. デザインを適用

### 方法2: 詳細版からカスタマイズ

### 推奨カラーパレット

```

アクセントカラー: #4A90A4 (ライトブルー)
強調カラー: #E8A87C (オレンジ)
背景: #FFFFFF (白) または #F9F9F9 (ライトグレー)
テキスト: #333333 (ダークグレー)
```

### 推奨フォント

- **日本語**: 游ゴシック Medium, メイリオ

- **サイズ**: タイトル 32-36pt / サブタイトル 24-28pt / 本文 20-24pt / 最小 18pt

1. **1スライド1メッセージ**
2. **箇条書きは3-5項目**
3. **十分な余白**

## プレゼンテーションのコツ

### 1. ストーリーテリング
- 課題から解決策へ自然な流れ
- 具体的な使用シーン
- データと感情的価値の両方

### 2. データと証拠
- 効果の数値（30-50%向上など）
- 導入前後の比較
- ケーススタディ

### 3. インタラクション
- 質問を促す、デモを実演、反応を見る

### 4. 時間管理
- 15スライド: 約20-30分
- 質疑応答: 10-15分
- デモ: 5-10分（オプション）

## よくある質問への準備

### Q1: セキュリティは大丈夫ですか？
- 暗号化（転送中・保存時）、個人情報保護法準拠、定期監査、アクセス制御

### Q2: コストはどのくらいですか？
- Basic: 月額5,000円 | Professional: 月額20,000円 | Enterprise: カスタム | 早期導入: 先着10施設50%オフ

### Q3: 導入は難しいですか？
- 専任サポート、トレーニング、無料トライアル（1-3ヶ月）、継続サポート

### Q4: 既存システムと統合できますか？
- API提供、カスタム統合、データエクスポート/インポート

### Q5: どのようなサポートがありますか？
- 初期設定、スタッフトレーニング、メール/電話/オンライン、定期アップデート
